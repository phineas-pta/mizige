# 米字格 (Mǐzìgé) — Chinese characters worksheet Generator

A web app for generating printable Chinese character practice worksheets with stroke order guides, pinyin annotations, and PDF export.

Built to help elementary students learn to write Chinese characters correctly.

forked from https://github.com/gelileo/mizige + deploy to github page

visit https://phineas-pta.github.io/mizige/ on **Google Chrome** to be able to download PDF

press Shift to toggle pinyin input method (status bar at bottom right of screen)
